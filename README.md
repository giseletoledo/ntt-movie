# ntt-movie
Projeto de consumo de API de filmes com Angular

Busca de filmes na API OMDB

## Funcionalidades

- Consumo de dados da API OMDB, endpoint de filme e detalhes
- Filtro de busca dos filmes favoritos

## Tecnologias Utilizadas

- Angular (Javascript)
- Git
- GitHub
